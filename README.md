# 📝 Notes App API

A beginner-friendly RESTful API built with **Node.js** and **Express.js** to create, read, update, and delete notes. This project demonstrates core backend development concepts including REST API design, middleware, routing, and file-based data persistence.

## Features
- Full CRUD operations (Create, Read, Update, Delete)
- RESTful API architecture with Express.js
- Custom request logging middleware
- File-based JSON storage (no external database required)
- Environment variable configuration with dotenv
- Centralized error handling and input validation

## Tech Stack
- Node.js
- Express.js
- File System (fs) module for data persistence
